## [0.1.0] - 2026-02-24

### Added

- Initial public release as a community project.
- Right-click "ZotExtract" context menu for Zotero items.
- Batch processing of multiple items with progress indicator.
- Configurable LLM provider (OpenAI, Ollama, any OpenAI-compatible API).
- Prompt Library with four built-in prompts:
  - Concise summary
  - Methods + data
  - Limitations
  - Figure/appendix scan (PDF file mode)
- Custom question support via prompt dialog.
- Two context modes: full-text extraction and raw PDF upload.
- Settings panel for provider configuration and prompt management.
- API key stored locally in Zotero profile.
- NFDIxCS branding and funding acknowledgement.
